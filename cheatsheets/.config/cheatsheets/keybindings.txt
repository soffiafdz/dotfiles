WM Modifier: Right side key (Right Cmd on Mac, Right Super on Linux)
             Feels like "Hyper" - same physical position everywhere

KEYBOARD REMAPPING
═══════════════════════════════════════════════════════════════
                        macOS (Karabiner)    Linux (xkb)
─────────────────────────────────────────────────────────────────
Caps Lock               Esc (tap) / Ctrl     Escape
Right Cmd/Super         Hyper → Aerospace    Super → dwm
Right Alt/Option        macOS compose        Compose key

WINDOW MANAGER (Hyper/Super + key)
═══════════════════════════════════════════════════════════════

FOCUS & NAVIGATION
─────────────────────────────────────────────────────────────────
  Focus (Aerospace)     Hyper + H/J/K/L      (spatial)
  Focus stack (dwm)     Super + J/K          (stack-based)
  Focus master (dwm)    Super + V
  Prev workspace        Hyper/Super + G
  Next workspace        Hyper/Super + ;
  Last workspace        Hyper/Super + Tab
  Switch to 1-9         Hyper/Super + 1-9
  View all              Hyper/Super + 0

LAYOUTS (dwm only)
─────────────────────────────────────────────────────────────────
  Tile                  Super + T
  Bstack                Super + Shift + T
  Monocle               Super + Y
  Deck                  Super + Shift + Y
  CenteredMaster        Super + U
  CFloatingMaster       Super + Shift + U
  nmaster +             Super + I
  nmaster -             Super + Shift + I

WINDOW ACTIONS
─────────────────────────────────────────────────────────────────
  Close window          Hyper/Super + Q
  Force kill (dwm)      Super + Shift + Q
  Fullscreen            Hyper/Super + F
  Float toggle          Hyper/Super + O
  Zoom/Layout cycle     Hyper/Super + Space
  Resize -              Hyper + - / Super + H
  Resize +              Hyper + = / Super + L

MOVE MODE (Hyper/Super + M, then...)
─────────────────────────────────────────────────────────────────
  Move window           H/J/K/L (Aerospace only)
  Send to workspace     1-9
  Send to prev ws       G (Aerospace follows to ws)
  Send to next ws       ; (Aerospace follows to ws)
  Send to prev monitor  P
  Send to next monitor  N
  Mount drive (dwm)     M
  Unmount drive (dwm)   U
  Cancel                Esc

TOGGLEVIEW MODE (dwm only: Super + C, then...)
─────────────────────────────────────────────────────────────────
  Toggle view tag       1-9

MONITORS
─────────────────────────────────────────────────────────────────
  Focus next            Hyper/Super + N
  Focus prev            Hyper/Super + P
  Send to next          Move mode + N / Super + Shift + N
  Send to prev          Move mode + P / Super + Shift + P

RAYCAST HOTKEYS (macOS - set up in Raycast Settings → Extensions)
═══════════════════════════════════════════════════════════════
  Hyper + S             Screenshot           (captures area)
  Hyper + Shift + S     Screenshot           (full screen - if available)
  Hyper + X             Clipboard History
  Hyper + Backspace     System Commands      (shutdown/restart/sleep)
  Hyper + Shift + D     1Password/Passwords  (like dmenupass)

TMUX (Ctrl-A prefix)
═══════════════════════════════════════════════════════════════
  Pane left/down/up/right    Prefix + H/J/K/L
  Resize pane                Prefix + Shift + H/J/K/L
  Split horizontal           Prefix + -
  Split vertical             Prefix + _
  Previous window            Prefix + Ctrl-H
  Next window                Prefix + Ctrl-L
  Last window                Prefix + Tab
  Copy mode                  Prefix + Enter

═══════════════════════════════════════════════════════════════
Note: "Hyper" on macOS = Cmd+Ctrl+Alt+Shift (via Karabiner)
      "Super" on Linux = Mod4 (the GUI/Win key)
      Same physical key (right side), same muscle memory.
